## Document Fraud: Assessing documents identities at susbcription

Subscription fraud is the first step of Fraud Committed. Fraudster suscribes at a platform app in the order to commit more serious and significant fraud. 
Firstly, models tables created allow to analyze in query mode or dashboard mode the state of subscriptions with documents ids.
Secondly, these models allow to model Machine Learning algorithms or SQL based-rules in the order to detect and prevent fraud.
